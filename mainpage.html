Hyperlinks

According to Google, hyperlink is a link from a hypertext file or document to another location or file. It is typically activate by clicking on a highlighted word or image on the screen.

Absolute References

Absolute reference is a type of link used in basicLinks.html. Absolute references always begin with the protocol name. An absolute reference is the complete address to a web page, just as you'd use in the browser's address bar. Absolute references are used to refer to a site somewhere else on the Internet.

Even if your website moves (say, from your desktop machine to a web server somewhere on the internet), all the absolute references will work fine because they don't rely on the current page's position for any information.

Example: https://www.dummies.com/web-design-development/html5-and-css3/absolute-and-relative-references-for-html5-and-css3-programming/

Relative References

Now, it's time for relative reference. Relative reference is a type of link mechanism for your website. For example, your website have several pages in them. You can link those pages so that the user can access it more easily using relative references.

When you click one of the links, you go to a brand-new page. That brand-new page should also contain a link back to the initial page using relative reference. Most websites aren't single pages at all, but an interconnected web of pages. The relative reference is very useful when you have a set of pages with interlacing links.

  • This one goes to CatPhotoApp

HTML Multimedia

These multimedia elements include many things, such as images, video, and audio. We can implement these things inside our website using tags like <video>, <audio>, and <img>

Here I will give you examples of each kind of multimedia.

Video

In here, I successfully implemented the video tag. As you can see, the video have control properties in it because we put an attribute called "controls" inside the video tag.

There are other attributes that can be used inside a video tag. These are href, class, controls, loop, autoplay, etc.

Please make sure that the video tag includes the control attribute to make the video more accessible and interactive to the user.

Audio

The next stop will be the audio tag. This tag, as the name implies, is used to display audio using html. The tag is simple, it is written like this <audio>.

Like the video tag, the audio tag has several attributes that can be used inside it. Such as, href, class, controls, loop, autoplay, buffered, etc.

Tables

We should only use tables if we have some kind of data being displayed. Please note that sketching your table beforehand is much easier than correcting wrong tables.

The tag that we use to make tables are simple. It consists of <table></table>. Inside the table tag, we have another tag to represent the rows and column. It should look something like this

<table>
<tr><td>One</td><td>Two</td><td>Three</td></tr>
<tr><td>Four</td><td>Five</td><td>Six</td></tr>
<tr><td>Seven</td><td>Eight</td><td>Nine</td></tr>
</table>
